# Simple Tkinter Animation 

Just a small Python project I made while learning Tkinter. It basically shows a circle moving across the screen...nothing fancy, just trying to understand how animations work in GUI.

---

## What it does
- Opens a basic window
- Draws a oval on the screen
- Moves it slowly to the right
- Keeps repeating so it looks like animation

---

## How it works (simple version)
It keeps calling the same function again and again using Tkinter’s `after()` method. That’s what makes the movement continuous instead of running once and stopping.
